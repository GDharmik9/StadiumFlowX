# StadiumFlow

**StadiumFlow** is an intelligent, full-stack Digital Twin simulation constructed specifically for the massive 63-acre Narendra Modi Stadium in Ahmedabad. 

The application bridges abstract venue blueprints seamlessly into a real-world GPS-constrained 3D environment, solving the "Scale Context" limitation frequently found in stadium applications. 

## Key Features
1. **Dynamic Spatial Digital Twin**: Runs an open-source 3D Vector engine (MapLibre GL JS) rendering massive geometric seating architectures procedurally over high-resolution ArcGIS satellite photography.
2. **Real-time Congestion Awareness**: Deploys an offline-first Firebase tracking architecture that pulses live "Occupancy Scores" mapping critical bottlenecks across Food Courts and Washrooms instantly.
3. **Multi-Level Topology Routing**: Utilizes IMDF-standard GeoJSON formatting natively allowing users to filter architectural Z-layers locally—burying concourse amenities directly beneath towering 35-meter seating extrusions! 
4. **Ring-Linked Wayfinding**: Enforces a non-linear pathfinding trajectory strictly forcing structural navigation along the 24-point walkable concourse perimeter.
